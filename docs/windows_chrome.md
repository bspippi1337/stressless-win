# Windows chrome (custom titlebar)

This WPF wrapper uses:
- `WindowStyle=None` and a custom titlebar
- caption buttons styled for an enterprise look
- best-effort **Mica/Acrylic** via DWM attributes (Win11 preferred)
- drag region + double-click maximize
- bottom-right resize grip

Files:
- `apps/windows-wpf/Stressless.Wpf/MainWindow.xaml`
- `apps/windows-wpf/Stressless.Wpf/MainWindow.xaml.cs`
- `apps/windows-wpf/Stressless.Wpf/Theme.xaml`
