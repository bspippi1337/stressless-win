# Media kit

Brand assets live in `branding/`.

## Recommended repo visuals
- README hero: `branding/hero.svg` or `branding/hero-compact.svg`
- Social preview: `branding/social-preview.png`
- App icon (png): `branding/icon-256.png`
- App icon (ico): `branding/stressless.ico`

## Palette
- Background: `#0A0C10`
- Surface: `#0F131A`
- Border: `#1C2430`
- Text: `#E8EEF7`
- Muted: `#9AA8BD`
- Primary: `#3B82F6` (use sparingly)
