# Stressless Brand Kit (alpha)

A compact, tasteful visual identity for **Stressless**.

## Palette
- **Midnight**: #0A0C10 (background)
- **Panel**: #0F131A
- **Line**: #1C2430
- **Text**: #E8EEF7
- **Muted**: #9AA8BD
- **Primary**: #3B82F6 (signal, not neon)
- **Success**: #22C55E
- **Danger**: #EF4444

## Tone
Quiet, technical confidence. No carnival gradients. Contrast that reads well for long sessions.

## Assets
- `logo.svg` wordmark + mark
- `mark.svg` standalone mark
- `icon-512.png` and `icon-256.png`
- `favicon-32.png`
